# Esketamine Brain

**Esketamine / Treatment-Resistant Depression (TRD) Daily Literature Report**

AI-powered PubMed surveillance system that automatically fetches, analyzes, and publishes daily Esketamine & TRD literature reports.

## Features

- Automated daily PubMed search across 57+ Esketamine-relevant journals (psychiatry, psychopharmacology, neuroscience, anesthesiology, pain)
- Keywords sourced from comprehensive esketamine medical journals PubMed templates
- AI analysis with Zhipu GLM-5-Turbo (fallback: GLM-4.7 → GLM-4.7-Flash)
- PICO analysis, clinical utility assessment, and topic classification
- Warm-toned HTML reports (matching Psychiatry-brain color scheme)
- Traditional Chinese (Taiwan) summaries for healthcare professionals

## Schedule

- Daily at 11:00 AM Taipei Time (UTC 03:00)
- Manual trigger available via GitHub Actions workflow_dispatch

## Topics Covered

- Esketamine / Spravato / S-ketamine
- Treatment-resistant depression (TRD)
- Suicidality and rapid-acting antidepressants
- NMDA / Glutamate / AMPA mechanisms
- Neuroplasticity and biomarkers
- Maintenance and relapse prevention
- Real-world effectiveness and implementation
- Perioperative / Anesthesiology / Pain applications
- Comparative ketamine vs esketamine studies
